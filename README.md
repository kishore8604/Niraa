# Niraa Front-End Demo

Static responsive e-commerce front-end for **Niraa – A unit of SP Impex**.

## Folder structure

```text
Niraa/
├── index.html
├── pages/
│   ├── account.html
│   └── bulk-order.html
├── css/
│   └── styles.css
├── js/
│   └── app.js
└── assets/
```

## Run locally

Open `index.html` directly, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
